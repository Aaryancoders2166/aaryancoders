What is HTML?<\/span><\/h3>\n

Okay, so this is the only bit of mandatory theory. In order to begin to write HTML, it helps if you know what you are writing.<\/p>\n

HTML is the language in which most websites are written<\/strong>. HTML is used to create pages and make them functional.<\/p>\n

The code used to make them visually appealing is known as CSS and we shall focus on this in a later tutorial. For now, we will focus on teaching you how to build rather than design<\/strong>.<\/p>\n

The History of HTML<\/span><\/h3>\n

HTML was first created by Tim Berners-Lee, Robert Cailliau, and others starting in 1989<\/strong>. It stands for Hyper Text Markup Language.<\/p>\n

Hypertext means that the document contains links that allow the reader to jump to other places<\/strong> in the document or to another document altogether. The latest version is known as HTML5.<\/p>\n

A Markup Language<\/strong> is a way that computers speak to each other to control how text is processed and presented. To do this HTML uses two things: tags and attributes<\/strong>.<\/p>\n

What are Tags and Attributes?<\/span><\/h3>\n

Tags and attributes are the basis of HTML.<\/p>\n

They work together but perform different functions \u2013 it is worth investing 2 minutes in differentiating the two<\/strong>.<\/p>\n

What Are HTML Tags?<\/h4>\n

Tags are used to mark up the start of an HTML element<\/strong> and they are usually enclosed in angle brackets. An example of a tag is: <h1><\/code>.<\/p>\n

Most tags must be opened <h1><\/code> and closed <\/h1><\/code> in order to function.<\/p>\n

What are HTML Attributes?<\/h4>\n

Attributes contain additional pieces of information<\/strong>. Attributes take the form of an opening tag and additional info is placed inside<\/strong>.<\/p>\n

An example of an attribute is:<\/p>\n

\n

<img src=\"mydog.jpg\" alt=\"A photo of my dog.\"><\/code><\/p>\n<\/div>\n

In this instance, the image source (src) and the alt text (alt) are attributes of the <img><\/code> tag.<\/p>\n

Golden Rules To Remember<\/h4>\n
    \n
  1. The vast majority of tags must be opened<\/strong> (<tag><\/code>) and closed<\/strong> (<\/tag><\/code>) with the element information such as a title or text resting between the tags.<\/li>\n
  2. When using multiple tags, the tags must be closed in the order in which they were opened<\/strong>. For example:\n

    <strong><em>This is really important!<\/em><\/strong><\/code><\/p>\n<\/li>\n<\/ol>\n

    Creating Your First HTML Webpage<\/span><\/h3>\n

    First off, you need to open your HTML editor, where you will find a clean white page<\/strong> on which to write your code.<\/p>\n

    From there you need to layout your page with the following tags.<\/p>\n

    Basic Construction of an HTML Page<\/h4>\n

    These tags should be placed underneath each other at the top of every HTML page<\/strong> that you create.<\/p>\n

    \n

    <!DOCTYPE html><\/code> \u2014 This tag specifies the language<\/strong> you will write on the page. In this case, the language is HTML 5.<\/p>\n

    <html><\/code> \u2014 This tag signals that from here on we are going to write in HTML code.<\/p>\n

    <head><\/code> \u2014 This is where all the metadata for the page<\/strong> goes \u2014 stuff mostly meant for search engines and other computer programs.<\/p>\n

    <body><\/code> \u2014 This is where the content of the page<\/strong> goes.<\/p>\n<\/div>\n
